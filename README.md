## Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Build for Production

```bash
npm run build
```

This builds the app for production to the `build` folder.

## Usage

1. Fill in the basic information (Entity Name and Email are required)
2. Add location details (optional)
3. Add social media links for any platforms you want to include
4. Click "Register Entity" to submit

The form validates all URLs and required fields before submission.
